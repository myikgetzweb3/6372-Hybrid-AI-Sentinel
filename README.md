## 🛠️ Quick Start

### 1. Requirements
*   Python 3.10+
*   Ollama (for local AI analysis)
*   Gemini API Key (for deep synergy analysis)

### 2. Installation
```bash
git clone https://github.com/myikgetzweb3/6372-Hybrid-AI-Sentinel.git
cd 6372-Hybrid-AI-Sentinel
pip install -r requirements.txt
```

### 3. Configuration
Rename `.env.example` to `.env` and fill in your credentials:
```bash
cp .env.example .env
nano .env
```

### 4. Deployment
Start the sentinel guardian:
```bash
bash start.sh
```
Monitor the dashboard:
```bash
python3 src/ui/dashboard.py
```
